# AI Trivia Arena — Brand Kit

## Identity

**Name:** AI Trivia Arena
**Tagline:** Every question is new. Every game is different.
**Voice:** Smart, competitive, playful. Like a trivia host who's also an AI researcher.

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| Electric Blue | `#3B82F6` | Primary CTA, links, highlights |
| Purple | `#8B5CF6` | Secondary accent, gradients |
| Gold | `#F59E0B` | Streaks, achievements, rank #1 |
| Green | `#10B981` | Correct answers, success |
| Red | `#EF4444` | Wrong answers, errors |
| Background | `#0f0f23` | Dark page background |
| Card | `#111827` | Card surfaces |

## Typography

- **Headings:** Geist Sans, bold/black weight
- **Body:** Geist Sans, regular
- **Code/Scores/Counters:** Geist Mono (monospace)
- **Font sizes:** Text-xl+ for questions, text-sm for metadata

## Dark Mode

Always dark mode. The app forces `className="dark"` on the `<html>` element.
Background: deep navy (`#0f0f23`). Not pure black — has a subtle blue tint.

## UI Patterns

- Glow orbs: radial gradients with 20-30% opacity for background depth
- Card borders: `border-border` default, `border-blue-500` on hover
- Gradient text: blue to purple to pink for headlines
- Timer: turns red + pulses at 5 seconds or less
- Score: Geist Mono, blue accent color
- Buttons: 44px+ min-height for touch targets
